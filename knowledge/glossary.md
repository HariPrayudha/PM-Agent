# Glossary PM — EN ↔ ID

> Untuk konsistensi terjemahan requirement client (EN) ↔ task tim (ID). Istilah teknis umum tidak perlu diterjemahkan kalau tim sudah terbiasa (mis. "deploy", "staging").

## Istilah PM

| EN | ID / Penjelasan | Catatan pemakaian |
|---|---|---|
| Scope | Cakupan pekerjaan | "In scope / out of scope" biasakan tetap EN ke tim |
| Scope creep | Pembengkakan scope diam-diam | Flag segera, jangan tunggu menumpuk |
| Deliverable | Hasil serahan | Konkret & bisa dicek |
| Milestone | Titik capaian | Harus bermakna bagi client |
| Timeline | Lini masa / jadwal | |
| Deadline | Tenggat | Bedakan: internal (tim) vs committed (client) |
| Buffer | Waktu cadangan | Milik PM, bukan waktu santai tim |
| Sign-off | Persetujuan formal | Selalu minta tertulis |
| Handover | Serah terima | |
| Change request (CR) | Permintaan perubahan di luar scope | Estimasi dulu → approval → kerjakan |
| Acceptance criteria (AC) | Kriteria diterima | Penentu "selesai"-nya task |
| Definition of Done (DoD) | Definisi selesai | Berlaku untuk semua task |
| UAT (User Acceptance Testing) | Uji terima oleh client | Ada periode, aturan, dan sign-off |
| Blocker | Penghambat | Prioritas #1 PM untuk diangkat |
| Stakeholder | Pemangku kepentingan | |
| Decision maker | Pengambil keputusan | Cari tahu sejak discovery |
| Kickoff | Rapat/fase mulai project | |
| Retro(spective) | Evaluasi pasca-project | Blameless |
| Lead time | Waktu dari permintaan → selesai | |
| Dependency | Ketergantungan | Termasuk dependensi ke client! |
| DPA (Digital PA) | Agent/perantara di sisi client SG yang menangani koordinasi dengan agency | Bukan istilah PM umum — istilah internal tim Hari. Komunikasi biasanya di grup WA bareng client. |

## Frasa yang sering muncul dari client SG

| Client bilang | Maksudnya biasanya |
|---|---|
| "Kindly revert" | Tolong balas/kabari (bukan revert git!) |
| "Please help to…" | Tolong kerjakan… (sopan, umum di SG) |
| "Can we do a quick call?" | Ada hal penting/tidak puas yang tidak mau ditulis — siapkan diri |
| "Just a small change…" | Belum tentu small — cek dampak dulu sebelum meng-iya-kan |
| "ASAP" | Minta tanggal spesifik: "We can have this by {date} — does that work?" |
| "Noted" | Sudah dibaca (belum tentu setuju — kalau butuh persetujuan, minta eksplisit) |
| "Revert by EOD" | Balas sebelum akhir hari kerja (SGT = WIB+1!) |

## Konversi waktu cepat

| WIB (Jakarta) | SGT (Singapura) |
|---|---|
| 08:00 | 09:00 |
| 12:00 | 13:00 |
| 17:00 | 18:00 |

Jam kerja overlap efektif: 09:00–17:00 WIB = 10:00–18:00 SGT.
